# How to run the project on your local system
 1. Clone the project 
 ```git clone https://github.com/namitaarya/garbage-man.git``` 

2. Run the frontend 
```cd client```
```npm install```
```npm start```

3. Start the backend server
```cd server```
```npm install```
```nodemon server.js```
